BYD Sealion 7 — Enable ADB & Install Linux

The Sealion 7 runs a locked-down DiLink Android infotainment system. Follow these exact steps to unlock ADB and sideload apps. Do this while parked.

1
Enable ADB on the BYD Sealion 7
On the car screen ▶

Follow these steps exactly on the car's touchscreen:

1
Open Settings
Tap the gear icon on the home screen or swipe down from the top bar.
2
Go to System
Scroll down and tap System in the settings list.
3
Tap Version
Inside System, tap Version to open the version info screen.
4
Press "Restore Factory" 10 times
Find the Restore Factory button and tap it 10 times rapidly. A hidden developer menu will appear. Do NOT confirm any factory reset prompt.
5
Rotate the screen
The hidden menu may require rotating the display orientation to reveal all options. Tap the rotate button if present.
6
Tap "Connect USB"
In the hidden developer menu, tap Connect USB. ADB is now enabled. Plug a USB cable from the car's data port to your laptop.
⚠ Do not tap "Restore Factory" and confirm — just tap it 10 times fast to trigger the hidden menu. Do not confirm any reset dialog.
💡 Tip: Use the car's USB-A data port (usually in the center console), not the USB-C charging port. If the PC doesn't detect the device, try a different USB cable or laptop port.

Then verify the connection from your PC:

adb devices

You should see the Sealion 7 listed as an ADB device. No RSA prompt will appear on the car screen — it connects silently.

3
Install Termux via ADB (sideload)
On PC ▶

Google Play's Termux is outdated. Install it from F-Droid via ADB sideload:

⚠ Warning: Do not install Termux from Google Play — it is no longer maintained there and will not receive package updates.

Download the latest Termux APK from F-Droid, then install it:

# Install Termux APK (replace filename with downloaded version) adb install -r com.termux_118.apk

Or use this shell to run the install command — go to the Install APK tab to sideload the APK file directly from your browser.

F-Droid link: https://f-droid.org/en/packages/com.termux/
4
Set up Linux inside Termux
On Android ▶

Open Termux on your device and run these commands one by one:

Update packages:

pkg update && pkg upgrade -y

Install proot-distro (Linux distro manager for Termux):

pkg install proot-distro -y

Install a Linux distro (choose one):

# Ubuntu (recommended for beginners) proot-distro install ubuntu # OR Debian proot-distro install debian # OR Alpine (very lightweight) proot-distro install alpine # OR Arch Linux proot-distro install archlinux

Enter (boot into) the distro:

proot-distro login ubuntu
Tip: You are now inside a full Ubuntu shell. You can run apt install, create users, install servers, Python, Node.js, etc.
5
Connect to Linux over ADB shell (remote access)
On PC ▶

You can drive Termux/Linux entirely from this ADB shell without touching the device screen.

Launch Termux in background via ADB:

adb shell am start -n com.termux/.HomeActivity

Run a command directly inside Termux via ADB:

adb shell run-as com.termux /data/data/com.termux/files/usr/bin/bash -c "your-command"

Set up SSH inside Termux so you can SSH in from your PC:

# Inside Termux on device: pkg install openssh -y sshd # starts SSH on port 8022 passwd # set a password # On your PC: adb forward tcp:8022 tcp:8022 ssh -p 8022 localhost
Tip: After adb forward, you get a full SSH session into Linux on your Android device from your PC terminal.

Optional — enable GUI desktop via VNC:

# Inside proot-distro Ubuntu: apt install xfce4 tigervnc-standalone-server -y vncserver -localhost no :1 # Forward VNC port from PC: adb forward tcp:5901 tcp:5901 # Connect VNC viewer to localhost:5901
